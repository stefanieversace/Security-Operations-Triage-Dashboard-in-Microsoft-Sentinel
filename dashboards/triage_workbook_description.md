# SOC Triage Dashboard Design

## Visual Components

### 1. Failed Login Spike Chart
- Time series of failed logins

### 2. Top Attacking IPs
- Bar chart of IPs with most failed attempts

### 3. Suspicious Successful Logins
- Table of high-frequency successful logins

### 4. Privileged Account Activity
- Recent privilege escalation events

### 5. Threat Intelligence Matches
- IPs matching known malicious indicators

---

## Purpose

This dashboard allows analysts to:
- Quickly identify anomalies
- Prioritise incidents
- Pivot into deeper investigation
