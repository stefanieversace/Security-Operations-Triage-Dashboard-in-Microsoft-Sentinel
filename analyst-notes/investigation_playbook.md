# Investigation Playbook

## Step 1: Validate Alert
- Confirm abnormal activity
- Check historical behaviour

## Step 2: Investigate User
- Login patterns
- Location anomalies
- Device usage

## Step 3: Investigate IP
- Threat intelligence lookup
- Frequency of activity

## Step 4: Check Privilege Changes
- Identify escalation events

## Step 5: Assess Lateral Movement
- Look for unusual internal connections

## Step 6: Response Actions
- Disable account
- Block IP
- Initiate incident response
